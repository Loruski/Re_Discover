# Re:Discover

An exploration based application which will help the user on visiting points of interest, historical places, and elements worth of note in any city they're interested about, in a way that will actively engage them by using notable aspects of software gamification.


## Main Functions
- Discover points of interest in any city around the world
- Get detailed information about each point of interest
- Gamification elements to enhance user engagement
- User-friendly interface for easy navigation
- Learn about the history and significance of various locations
- Interactive maps to guide users to points of interest

### Gamification Features
- **Achievement Badges**: Earn badges for visiting multiple points of interest or completing specific challenges
- **Leaderboards**: Compete with friends or other users based on points earned from visits
- **Points System**: Earn points for each point of interest visited
- **Quizzes**: Participate in quizzes related to the points of interest to earn points
- **Personalization**: Customize your profile and track your progress
